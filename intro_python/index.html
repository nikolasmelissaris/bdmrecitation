Downloading and installing Python 3 on your computer

If you are using a Mac simply head over to the Python website /a> and download the most recent package. Then double-click on the downloaded file to run the Python 3 installer.

If you are using Windows as a first step, download the suitable installer from the Python website /a>. Double-click on the installer and on the first screen, check the “Add Python 3.x to PATH” option and install.
